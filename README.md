# Mutenge's Fast Food

This is a simple user site for Mutenge's Fast Food. The site is built with static HTML, CSS, and JavaScript and is intended to be published via GitHub Pages at https://maybinmutenge-web.github.io/

To edit locally:

```bash
# clone
git clone git@github.com:maybinmutenge-web/maybinmutenge-web.github.io.git
cd maybinmutenge-web.github.io
# serve locally
python -m http.server 8000
```
